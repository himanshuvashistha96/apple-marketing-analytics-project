## Apple Marketing Analytics Project

## Overview
This project analyzes Apple’s social media marketing performance using Excel-based analytics techniques.

## Objectives
- Analyze engagement metrics
- Track campaign ROI
- Evaluate platform performance
- Identify high-performing content

## Tools Used
- Microsoft Excel
- Pivot Tables
- VLOOKUP
- SUMIFS
- IF/IFS
- Charts & Dashboards

## Key Insights
- Video content generated the highest engagement
- Instagram had strongest follower growth
- Some high-spend campaigns produced lower ROI

## Files Included
- Raw dataset
- Cleaned dataset
- Analysis tasks
- Dashboard reports

- Data Cleaning
- Data Analysis
- KPI Reporting
